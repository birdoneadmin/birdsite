เกี่ยวกับ MonteSmart

MonteSmart เป็นศูนย์การศึกษาออนไลน์ที่ฟรีสำหรับทุกคน เรายินยอมรับทุกคนตั้งแต่เด็กถึงผู้ใหญ่ด้วยคอร์สที่ไม่มีค่าใช้จ่าย

เราสอนข้อมูลที่เป็นประโยชน์เกี่ยวกับเทคโนโลยีตั้งแต่การสร้างเว็บไซต์ไปจนถึงการใช้งาน AI

Start Creating

คอร์สเรียน Start Creating เหมาะสำหรับผู้เริ่มต้นที่ต้องการเรียนรู้การใช้โปรแกรมต่างๆ เรามีสิ่งที่เป็นประโยชน์เช่นการสร้างสไลด์ การสร้างเว็บไซต์ การเขียนโค้ดสำหรับผู้เริ่มต้น และการใช้งาน AI พื้นฐาน

วันเรียน: ทุกวันอาทิตย์ตั้งแต่ 21 พฤษภาคม 2566 - 9 กรกฎาคม 2566

พื้นฐานภาษา Python

ในคอร์สพื้นฐานภาษา Python เราสอนการเริ่มต้นโปรแกรมโปรเจคที่สนุกโดยใช้ภาษา Python เราเริ่มต้นง่าย และเพิ่มความซับซ้อนในแต่ละตอน
